# E-Commerce Sales Analysis Dashboard

## Project Overview
This project is an interactive E-Commerce Sales Analysis Dashboard created using Power BI. The dashboard helps analyze sales data, revenue trends, customer behavior, product performance, and business growth insights.

## Objective
The main objective of this project is to understand e-commerce sales performance and generate useful insights for business decision-making.

## Tools Used
- Power BI
- SQL
- Microsoft Excel
- Data Cleaning
- Data Visualization

## Key Insights
- Revenue and sales trend analysis
- Product-wise performance
- Customer behavior analysis
- Category-wise sales insights
- Business performance KPIs

## Skills Demonstrated
- SQL Queries
- Data Cleaning
- Data Analysis
- Data Visualization
- Dashboard Development
- Power BI Reporting

## Project Outcome
Created a clear and interactive dashboard to present e-commerce sales insights in a simple and meaningful way.
